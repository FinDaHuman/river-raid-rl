# River Raid RL Documentation

Welcome to the River Raid RL documentation. This project implements a **Rainbow DQN** agent that plays the Atari 2600 game **River Raid** using deep reinforcement learning.

## Quick Navigation

- **[Getting Started](getting-started.md)** – Install, configure, and train an agent.
- **[Agent Architecture](architecture.md)** – Network design, replay buffer, training loop.
- **[Ablation Study](ablation.md)** – How to run component‑wise ablations.
- **[API Reference](api.md)** – Public classes and functions.

## Project Goals

- Reproducible RL research with YAML‑driven experiments.
- Low‑end hardware support (CPU laptop).
- Full ablation study across all Rainbow components.
